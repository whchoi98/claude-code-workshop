## Speaker Notes

Pattern 5 매일 보고서 설계입니다.
동작 흐름은 6단계입니다.
오전 9시에 트리거되어 어제 활동을 수집하고 카테고리 분류, AI 요약, Markdown 포맷, Slack 게시 순서로 진행됩니다.
혜택은 명확합니다.
팀 stand-up 회의 전에 어제 활동을 자동 요약해 공유합니다.
사람은 자신이 한 일을 다시 적을 필요 없이 이미 정리된 내용을 검토만 합니다.
수집되는 데이터는 6가지입니다.
첫째, 머지된 PR 목록과 카테고리입니다.
둘째, 클로즈된 이슈와 해결 방법입니다.
셋째, 새로 열린 이슈와 우선순위입니다.
넷째, 활발한 contributor 통계입니다.
다섯째, 빌드와 배포 상태입니다.
여섯째, 모니터링 알람 발생 여부입니다.
팀 활동의 전체 그림을 매일 자동으로 파악할 수 있습니다.
