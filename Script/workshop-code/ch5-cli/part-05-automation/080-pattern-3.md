## Code Blocks

### safe migration

```bash
# 안전한 마이그레이션 흐름
1. 변환 규칙 정의 (한 번)
2. 작은 샘플 (5-10 파일)로 검증
3. 결과 확인 후 본격 변환
4. 각 파일 변환 후 즉시 테스트
5. 실패 시 자동 롤백 + 사람 검토
6. 전체 완료 후 통합 테스트
```

## Speaker Notes

Pattern 3 코드 마이그레이션 설계입니다.
사용 사례를 먼저 봅니다.
Vue 2에서 Vue 3 마이그레이션, jQuery에서 React, JS에서 TS, lodash 제거 같은 대규모 변환에 적합합니다.
도전 과제는 명확합니다.
수백 파일을 동시에 변환해야 하고 일관성을 유지해야 하며 안전한 검증과 부분 실패 복구가 필요합니다.
접근 방식은 AI와 사람의 협업입니다.
AI가 변환을 수행하고 자동 테스트로 검증합니다.
사람은 의심스러운 결과만 검토합니다.
안전한 마이그레이션 흐름은 6단계입니다.
1단계 변환 규칙을 한 번 명확히 정의합니다.
2단계 5에서 10개의 작은 샘플로 먼저 검증합니다.
3단계 결과 확인 후 본격 변환을 시작합니다.
4단계 각 파일 변환 후 즉시 테스트를 실행합니다.
5단계 실패 시 자동 롤백하고 사람 검토를 요청합니다.
6단계 전체 완료 후 통합 테스트를 수행합니다.
이 흐름이 안전한 자동 마이그레이션의 표준입니다.
