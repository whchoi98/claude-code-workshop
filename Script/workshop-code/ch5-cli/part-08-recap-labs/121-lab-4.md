# Slide 121: Lab 4 / 데이터 파싱

**Part 8: RECAP & 4 LABS**

## Code Blocks

### Lab 4

```bash
#!/bin/bash
# generate-dashboard.sh - 응답 데이터 시각화

mkdir -p data

# Step 1: 여러 호출 실행 (데이터 수집)
for topic in "Python" "JavaScript" "Go" "Rust" "TypeScript"; do
  claude -p "$topic 언어의 장점 3가지" \
    --output-format json > "data/$topic.json"
done

# Step 2: JSON → CSV 변환
{
  echo "language,duration_ms,input_tokens,output_tokens,model"
  jq -rs '.[] | [
    input_filename | sub("data/"; "") | sub(".json"; ""),
    .duration_ms,
    .usage.input_tokens,
    .usage.output_tokens,
    .model
  ] | @csv' data/*.json
} > stats.csv

cat stats.csv

# Step 3: 통계 계산
echo ""
echo "=== 통계 ==="
jq -rs '
  {
    total_calls: length,
    avg_duration_ms: ([.[].duration_ms] | add / length | floor),
    total_input: ([.[].usage.input_tokens] | add),
    total_output: ([.[].usage.output_tokens] | add)
  }
' data/*.json | jq .

# Step 4: HTML 대시보드 생성
DATA=$(jq -rs '...' data/*.json)
cat <<HTML > dashboard.html
<!DOCTYPE html>
<html><head>
<title>Claude Usage Dashboard</title>
<style>
body { font-family: sans-serif; max-width: 800px; margin: auto; }
table { border-collapse: collapse; width: 100%; }
th, td { padding: 8px; border-bottom: 1px solid #ddd; }
th { background: #f0f0f0; }
</style>
</head><body>
<h1>📊 Claude API 사용 통계</h1>
<p>생성: $(date)</p>
$(cat stats.csv | awk -F, 'BEGIN{print "<table>"}
  NR==1{print "<tr><th>"$1"</th><th>"$2"</th></tr>"; next}
  {print "<tr><td>"$1"</td><td>"$2"</td></tr>"}
  END{print "</table>"}')
</body></html>
HTML

echo "✓ dashboard.html 생성 완료"
echo "  열기: open dashboard.html (macOS) / start dashboard.html (Windows)"
```

## Speaker Notes

Lab 4는 데이터 파싱 실습입니다.
약 20분 소요됩니다.
JSON 응답을 CSV로 변환하고 HTML 대시보드를 생성합니다.
Step 1에서 여러 호출을 실행해 데이터를 수집합니다.
5가지 프로그래밍 언어에 대한 응답을 별도 파일로 저장합니다.
Step 2에서 JSON을 CSV로 변환합니다.
헤더를 먼저 출력하고 jq -rs로 모든 파일을 배열로 합친 후 각 응답을 CSV 행으로 변환합니다.
input_filename에서 언어 이름을 추출합니다.
Step 3에서 통계를 계산합니다.
총 호출 수, 평균 응답 시간, 총 입력과 출력 토큰을 jq로 계산합니다.
Step 4에서 HTML 대시보드를 생성합니다.
heredoc으로 HTML 골격을 만들고 CSV를 awk로 HTML 테이블로 변환해 삽입합니다.
DOCTYPE, head, body, style을 포함한 완전한 HTML이 생성됩니다.
open 명령으로 브라우저에서 열어 확인할 수 있습니다.
자동 생성된 대시보드를 직접 보면서 데이터 파싱의 효과를 체감할 수 있습니다.
