```bash

# 2. 생성된 CLAUDE.md를 본인 프로젝트에 맞춰 편집
```

## Speaker Notes

Lab 2는 CLAUDE.md 작성입니다.
약 20분 소요됩니다.
1단계 본인 프로젝트 루트에서 claude를 실행하고 /init 명령으로 자동 생성을 시도합니다.
2단계 생성된 CLAUDE.md를 본인 프로젝트에 맞춰 편집하는데, Project Overview, Tech Stack, Commands, Conventions, Architecture 같은 핵심 섹션을 작성합니다.
화면 예시처럼 한국어 1-2 문장으로 프로젝트 목적을 설명하고, 기술 스택과 주요 명령어, 코딩 컨벤션을 명시합니다.
3단계 새로운 세션을 시작해 CLAUDE.md가 자동으로 로드되어 작업이 더 정확해지는지 확인합니다.
새 endpoint 추가를 요청해 보면 본인 프로젝트의 컨벤션을 자동으로 따르는지 관찰할 수 있습니다.
