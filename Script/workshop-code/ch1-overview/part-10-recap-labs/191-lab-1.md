## Code Blocks

### Lab 1

```bash
# 단계별 명령
$ node --version              # v18+ 확인
$ npm install -g @anthropic-ai/claude-code
$ claude --version
$ claude doctor               # 모든 ✓ 확인
$ cd ~/your-test-project
$ claude
> "이 프로젝트의 구조를 간단히 설명해 주세요"

# 성공 기준
  ✓ doctor 통과   ✓ 인증 완료   ✓ 첫 응답 수신   ✓ Read 도구 호출 관찰
```

## Speaker Notes

Lab 1은 설치와 첫 세션 시작입니다.
약 15분 소요됩니다.
5단계로 진행하는데, 1단계 node와 git, IDE 같은 사전 환경을 점검하고, 2단계 본인 환경에 맞는 방법으로 설치합니다.
3단계 claude doctor를 실행해 모든 항목에 체크 표시가 나오는지 확인합니다.
4단계 /login 명령 또는 환경변수로 API Key를 설정해 인증을 마칩니다.
5단계 본인의 테스트 프로젝트로 이동해 claude 명령으로 첫 세션을 시작하고, 간단한 프롬프트로 응답을 받아 봅니다.
성공 기준은 doctor 통과, 인증 완료, 첫 응답 정상 수신, Read 도구 호출이 관찰되는 것입니다.
어려움이 있다면 즉시 멘토에게 도움을 요청하시기 바랍니다.
