# Slide 194: Lab 4

**Part 10: RECAP & LABS**

## Code Blocks

### Lab 4

```bash
# 1. claude 시작 후 Plan 모드 진입
$ claude
> [Shift+Tab]                 # Plan 모드 토글
# 또는 처음부터:
$ claude --plan

# 2. 복잡한 작업을 요청
> 사용자 모델에 soft delete 기능을 추가해 주세요.
> deleted_at 컬럼, 미들웨어, 마이그레이션, 테스트를 모두 포함해야 합니다.

# 3. Plan 검토
#   - Claude가 단계별 계획 제시
#   - 영향 받는 파일 목록
#   - 변경 요약과 위험 분석

# 4. 승인 또는 수정
> 좋은데, 마이그레이션은 분리해 별도 PR로 만들어 주세요.

# 5. 실행 결과 검토 및 Git 커밋
```

## Speaker Notes

Lab 4는 Plan 모드 활용 실습입니다.
약 20분 소요됩니다.
1단계 claude를 시작한 후 Shift+Tab으로 Plan 모드로 진입하거나 --plan 플래그로 처음부터 Plan 모드로 시작합니다.
2단계 복잡한 작업을 요청합니다.
사용자 모델에 soft delete 기능을 추가하면서 deleted_at 컬럼, 미들웨어, 마이그레이션, 테스트를 모두 포함해야 한다는 요구사항을 제시합니다.
3단계 Claude가 제시하는 단계별 계획과 영향 받는 파일 목록, 변경 요약, 위험 분석을 검토합니다.
4단계 필요하다면 수정 요청을 합니다.
마이그레이션은 분리해서 별도 PR로 만들어 달라고 요청할 수 있습니다.
5단계 합의된 계획대로 실행된 결과를 검토하고 Git 커밋과 PR을 생성합니다.
