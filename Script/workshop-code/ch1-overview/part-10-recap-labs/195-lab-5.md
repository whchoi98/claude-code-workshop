# Slide 195: Lab 5

**Part 10: RECAP & LABS**

## Code Blocks

### Lab 5

```bash
# 1. 간단한 헤드리스 호출
$ claude -p "최근 git log 5개를 요약해 주세요" --output-format text

# 2. JSON 출력으로 후처리
$ claude -p "package.json 의존성 분석" --output-format json | jq '.summary'

# 3. 쉘 파이프 통합
$ cat error.log | claude -p "이 에러 로그를 분류해 주세요"

# 4. 자동화 스크립트 예시
$ cat > review.sh << 'EOF'
#!/bin/bash
DIFF=$(git diff HEAD~1)
echo "$DIFF" | claude -p "이 PR diff를 검토해 주세요" \
  --allowed-tools "" --output-format text > review.md
EOF

# 5. cron 또는 GitHub Actions로 등록 (Chapter 5에서 심화)
```

## Speaker Notes

마지막 Lab 5는 Headless 자동화 실습입니다.
약 25분 소요됩니다.
1단계 -p 플래그로 간단한 헤드리스 호출을 시도합니다.
최근 git log 5개를 요약해 달라는 요청 같은 단순한 것부터 시작합니다.
2단계 --output-format json 옵션으로 구조화된 출력을 받고 jq 같은 도구로 후처리할 수 있는지 확인합니다.
3단계 stdin으로 입력을 파이프하여 쉘 파이프 통합이 가능한지 확인합니다.
error log를 분류해 달라는 요청을 보냅니다.
4단계 간단한 자동화 스크립트를 작성합니다.
git diff를 받아 PR 리뷰 코멘트를 생성하는 review.sh 같은 스크립트를 만듭니다.
5단계 cron이나 GitHub Actions로 등록해 진정한 자동화를 만드는데, 이 부분은 Chapter 5 CLI Reference에서 깊게 다룹니다.
