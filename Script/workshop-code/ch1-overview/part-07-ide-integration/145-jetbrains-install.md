# Slide 145: JetBrains install

**Part 7: IDE INTEGRATION**

## Code Blocks

### JetBrains

```bash
# 1. 플러그인 저장소에서 검색
File → Settings → Plugins (또는 Preferences on macOS)
Marketplace 탭 → "Claude Code" 검색
Install → IDE 재시작

# 2. 지원 IDE (동일 플러그인)
- IntelliJ IDEA (Ultimate / Community)
- PyCharm (Pro / Community)
- WebStorm
- GoLand
- RubyMine
- PhpStorm
- CLion
- Rider
- DataGrip (제한적)
- Android Studio

# 3. 인증
Tools → Claude Code → Sign In
- 브라우저 OAuth 또는 API Key

# 4. Tool Window 활성화
View → Tool Windows → Claude Code (Alt+9 권장)
```

## Speaker Notes

JetBrains 플러그인은 모든 IDE에서 동일하게 작동합니다.
1단계 Settings에서 Plugins로 진입해 Marketplace 탭에서 Claude Code를 검색하고 Install 후 IDE를 재시작합니다.
2단계 지원 IDE는 IntelliJ IDEA, PyCharm, WebStorm, GoLand, RubyMine, PhpStorm, CLion, Rider, DataGrip, Android Studio 등 거의 모든 JetBrains IDE입니다.
동일한 플러그인이 모든 IDE에서 작동합니다.
3단계 Tools 메뉴의 Claude Code Sign In으로 인증을 진행합니다.
4단계 View Tool Windows에서 Claude Code Tool Window를 활성화하는데 Alt 9 단축키로 빠르게 토글할 수 있습니다.
