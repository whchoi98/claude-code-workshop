# Slide 140: VS Code install

**Part 7: IDE INTEGRATION**

## Code Blocks

### VS Code Install

```bash
# 1. VS Code Marketplace 검색
View → Extensions (Ctrl/Cmd+Shift+X)
Search: "Claude Code"
Publisher: Anthropic

# 2. 또는 CLI로 설치
code --install-extension anthropic.claude-code

# 3. 명령 팔레트 (Cmd/Ctrl+Shift+P)
> Claude Code: Open Sidebar
> Claude Code: Start Session
> Claude Code: Sign In

# 4. 인증
- Sign In 명령 실행
- 브라우저 OAuth 또는 API Key 입력

# 5. 사이드바 활성화
- Activity Bar의 Claude 아이콘 클릭
- 또는 Ctrl/Cmd+Shift+L
```

## Speaker Notes

VS Code 확장 설치 과정입니다.
1단계 View 메뉴에서 Extensions를 열거나 Ctrl Shift X 단축키로 Marketplace를 열고 Claude Code를 검색합니다.
Publisher는 Anthropic입니다.
2단계 또는 명령행에서 code 명령에 install-extension 옵션을 주어 설치할 수도 있습니다.
3단계 설치 후 명령 팔레트 Cmd Shift P에서 Claude Code 관련 명령을 사용합니다.
4단계 Sign In 명령으로 인증을 진행하는데 브라우저 OAuth 또는 API Key 입력 중 선택할 수 있습니다.
5단계 Activity Bar의 Claude 아이콘이나 Ctrl Cmd Shift L 단축키로 사이드바를 활성화합니다.
