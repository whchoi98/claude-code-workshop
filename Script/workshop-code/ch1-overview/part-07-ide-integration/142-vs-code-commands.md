# Slide 142: VS Code commands

**Part 7: IDE INTEGRATION**

## Code Blocks

### commands

```bash
# 명령 팔레트 (Cmd/Ctrl+Shift+P) 주요 명령

> Claude Code: Open Sidebar              # 사이드바 토글
> Claude Code: Start New Session          # 새 세션
> Claude Code: Continue Last Session      # 마지막 세션 이어가기
> Claude Code: Send Current Selection     # 선택 영역 전송
> Claude Code: Send Current File          # 활성 파일 전송
> Claude Code: Toggle Plan Mode           # Plan 모드 토글
> Claude Code: Show Cost                  # 사용량 표시
> Claude Code: Sign Out / Sign In         # 인증 관리
> Claude Code: Open Settings              # 확장 설정 열기

# 컨텍스트 메뉴 (편집기에서 우클릭)
- Ask Claude about this file
- Refactor selection with Claude
- Explain this code
- Generate tests for this function
```

## Speaker Notes

VS Code 명령 팔레트의 주요 Claude Code 명령들을 정리했습니다.
Open Sidebar는 사이드바 토글, Start New Session은 새 세션 시작, Continue Last Session은 마지막 세션 이어가기, Send Current Selection은 선택 영역을 Claude에 전송, Send Current File은 활성 파일 전체 전송, Toggle Plan Mode는 Plan 모드 토글, Show Cost는 사용량 표시, Sign Out과 Sign In은 인증 관리, Open Settings는 확장 설정 열기 명령입니다.
편집기에서 우클릭하면 컨텍스트 메뉴로 Ask Claude about this file, Refactor selection, Explain this code, Generate tests 같은 빠른 작업이 제공됩니다.
