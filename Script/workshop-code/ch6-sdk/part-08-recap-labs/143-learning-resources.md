## Speaker Notes

Chapter 6 학습 자원을 정리합니다.
Official Documentation은 docs.anthropic.com을 참고합니다.
messages, tool-use, prompt-caching 같은 핵심 페이지를 자주 봅니다.
SDK Repositories는 GitHub의 anthropic-sdk-python과 anthropic-sdk-typescript입니다.
실제 SDK 코드와 examples 폴더를 참고할 수 있습니다.
Cookbooks가 매우 유용합니다.
anthropic-cookbook 저장소에 Tool use 패턴, Vision 활용, Multi-agent, RAG 통합 같은 실제 예시들이 있습니다.
비동기와 백그라운드 처리는 Celery, FastAPI, asyncio 공식 문서를 참고합니다.
관측성은 OpenTelemetry, structlog, CloudWatch 문서를 봅니다.
벡터 DB는 semantic 캐시 구현에 필요합니다.
Pinecone, Qdrant, pgvector 중 선택합니다.
테스트는 pytest, Locust, unittest.mock 문서를 참고합니다.
커뮤니티는 Anthropic Discord, Stack Overflow anthropic 태그, GitHub Discussions을 활용합니다.
사내 자산도 적극 활용하시기 바랍니다.
사내 wiki와 Slack 채널에 추가로 질문하고 답변을 얻을 수 있습니다.
