# Slide 116: FAQ

**Part 9: RECAP & 3 LABS**

## Code Blocks

### FAQ

```
Q1. 사내망에서 외부 인터넷 차단인데 Claude Code 사용 가능한가요?
A.  네, Bedrock VPC Endpoint를 통해 AWS 사설망 내에서만 통신 가능합니다.
    npm 미러는 사내에 구축, MCP 서버도 사내로.

Q2. Claude Code 도입 비용 예산을 어떻게 잡아야 하나요?
A.  1인당 월 $27 기준 + 30% 안전 마진. Opus 사용은 별도 통제.
    PoC 단계에서 실측 후 본 도입 시 조정 권장.

Q3. SSO와 OAuth Pro 중 선택은?
A.  10인 이상은 Bedrock + IAM Identity Center 권장.
    1-5인 PoC라면 OAuth Pro로 간단히 시작 가능.

Q4. 개발자가 managed-settings를 우회할 수 있나요?
A.  /etc/claude 디렉토리는 root 권한 필요. 일반 사용자는 불가능.
    deny 규칙은 항상 승리하므로 우회 방법 없음.

Q5. 컴플라이언스 감사 자료를 어디서 얻나요?
A.  CloudTrail (필수), OpenTelemetry (선택), managed-settings 문서.
    Anthropic에서 SOC 2 보고서 요청 가능.
```
