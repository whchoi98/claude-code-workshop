## Speaker Notes

Lab 2는 도구 권한 격리 실습입니다.
약 15분 소요됩니다.
안전한 에이전트를 만드는 패턴을 직접 체험합니다.
1단계 의도적으로 위험한 도구 사용을 시도하는 테스트 에이전트를 작성합니다.
dangerous-test 에이전트는 tools에 Read와 Grep만 허용했습니다.
Edit이나 Write는 없습니다.
2단계 일부러 권한을 벗어난 요청을 합니다.
비밀번호 부분을 수정해 달라고 합니다.
3단계 결과를 관찰합니다.
Edit 도구가 tools에 없으므로 거부됩니다.
권한 부족 메시지로 안전 동작을 확인합니다.
4단계 tools 필드를 수정해 Edit을 추가하고 재시도합니다.
이번에는 동작하지만 이는 명시적 권한 부여의 결과입니다.
암묵적으로 모든 권한이 부여되는 것과의 차이를 체험합니다.
