# Slide 103: Lab 1

**Part 9: RECAP & LABS**

## Code Blocks

### Lab 1 steps

```bash
# 1. 디렉토리 생성
$ mkdir -p .claude/agents

# 2. code-reviewer.md 작성 (Part 2 슬라이드 31 참고)
$ vim .claude/agents/code-reviewer.md

# 3. 등록 확인
$ claude
> /agents
  code-reviewer (PR diff 검토)

# 4. 첫 호출
> 다음 git diff를 검토해 주세요: <diff>...</diff>

# 5. 결과 검토
- 출력 형식이 적절한가
- 우선순위가 명확한가
- 본인 프로젝트 컨벤션을 따르는가
```

## Speaker Notes

Lab 1은 첫 Subagent를 작성하고 호출하는 실습입니다.
약 20분 소요됩니다.
5단계로 진행됩니다.
1단계 프로젝트 루트에 .claude/agents 디렉토리를 만듭니다.
2단계 Part 2의 슬라이드 31 예시를 참고해 code-reviewer.md 파일을 작성합니다.
frontmatter 4필드와 시스템 프롬프트 본문을 모두 포함합니다.
3단계 claude를 시작해 /agents 명령으로 등록을 확인합니다.
4단계 첫 호출을 합니다.
실제 git diff를 prompt 안에 넣어 검토를 요청합니다.
5단계 결과를 검토합니다.
출력 형식이 적절한지, 우선순위가 명확한지, 본인 프로젝트 컨벤션을 따르는지를 점검합니다.
결과가 만족스럽지 않으면 시스템 프롬프트를 수정하고 다시 호출합니다.
