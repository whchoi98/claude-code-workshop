## Speaker Notes

Lab 3는 Task 디스패치 실습입니다.
약 25분 소요됩니다.
3가지 호출 패턴을 직접 실습합니다.
사전에 code-reviewer, test-writer, docs-writer 세 에이전트가 등록되어 있다고 가정합니다.
패턴 A는 자동 디스패치입니다.
이 PR을 코드 리뷰해 달라고 하면 메인이 description을 보고 code-reviewer를 자동 선택하는 것을 확인합니다.
패턴 B는 수동 명시 호출입니다.
@test-writer로 직접 호출하면 메인이 다른 에이전트 검토 없이 즉시 호출합니다.
패턴 C는 병렬 호출입니다.
종합 점검 요청 시 메인이 3개 Subagent를 동시에 호출하는 것을 관찰합니다.
마지막으로 /cost 명령으로 비용을 비교합니다.
패턴 A는 단일 호출이라 약 0.05달러입니다.
패턴 C는 3개 병렬이라 약 0.18달러로 3배 비용이지만 시간은 1.5배만 늘어납니다.
비용과 시간의 트레이드오프를 직접 체험합니다.
