# Slide 99: Cost efficiency

**Part 8: PATTERN 5 / MIGRATION BOT**

## Code Blocks

### cost calc

```bash
# 시나리오: 200개 파일에서 동일한 패턴 변경

# Sonnet 사용 시
- 파일당 input 2K + output 0.5K
- 비용 = 200 × (2K × $3 + 0.5K × $15) / 1M
- = 200 × $0.0135 = $2.70

# Haiku 사용 시
- 같은 작업, 같은 토큰 사용량
- 비용 = 200 × (2K × $1 + 0.5K × $5) / 1M
- = 200 × $0.0045 = $0.90

# 절감 효과: $2.70 → $0.90 (-67%)

# 품질 영향
- 단순 패턴 매칭/교체 작업
- 컨텍스트 이해 필요 없음
- Haiku로도 충분한 품질
- 복잡한 추론이 필요한 작업이 아니라면 Haiku 적극 권장
```
